### 방법 2: Excel Online이나 Google Sheets 사용

1. 엑셀 파일을 Excel Online이나 Google Sheets에서 열기
2. '사업부' 컬럼으로 필터링 (MLB, MLB Kids, Discovery, 공통)
3. 각 브랜드별로 CSV로 다운로드
4. 온라인 CSV to JSON 변환기 사용 (예: convertcsv.com)
5. 변환된 JSON을 `public/data/cost_data.json`에 붙여넣기

### 방법 3: 제가 샘플 데이터 생성

제가 실제 엑셀 구조를 파악했으니, 샘플 데이터를 생성해드릴 수 있습니다.

## 📊 데이터 구조

엑셀 파일 구조:
- **T열 (사업부)**: MLB, MLB Kids, Discovery, 공통
- **B열 (Cost ctr desc)**: 본부명
- **U열 (부서명)**: 팀명
- **V열 (대분류)**: 계정과목
- **E~P열 (202401~202412)**: 2024년 월별 금액
- **E~N열 (202501~202510)**: 2025년 월별 금액
